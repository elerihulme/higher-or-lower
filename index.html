Higher or Lower?

Objective:

The goal of the game is to correctly guess whether the next card drawn will be higher or lower than the current card. If you successfully guess through 5 cards, you win the game. However, if you guess incorrectly at any point, the game resets and you have to start over.

Game Set Up

  1. The game starts with a single card displayed on the screen.
  2. Four additional card slots are shown, but they are initially hidden (displayed as ?).

How to Play

  1. Start the Game: When the game begins, you'll see the first card (e.g., a 7). The next card is hidden.
  2. Make a guess:
    • You have two buttons: "Higher" and "Lower".
    • Based on the value of the current card, you must guess whether the next card drawn will be higher or lower in value.
  3. Reveal the Next Card:
    • After making your guess, the next card is revealed.
    • If your guess is correct (e.g., the next card is indeed higher if you guessed "Higher"), you move on to the next round.
    • If your guess is incorrect, the game ends, and all cards reset to ?. You must start over from the first card.
  4. Continue Guessing:
    • Repeat the guessing process for the remaining cards. You need to guess correctly for each new card drawn.
    • The game keeps track of your progress through the 5 cards.

Winning the Game:

If you successfully guess correctly for all 5 cards, you win the game! A message will be displayed, and the game will reset to allow you to play again.

Losing the Game:

If you make an incorrect guess at any point, the game will display a "You Lose!" message, and the cards will reset. You must start from the first card again.
